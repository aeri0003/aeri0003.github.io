HEADING

Heading
We can use the h1 tag for the most important heading of the page. For the next level of heading we have h2, h3, h4, h5, and h6. The browser will style each heading differently, starting with the biggest size for h1 then h2 and so on.
Paragraphs
For adding a paragraph of text we can use the p tag. The browser will add a space before and after the content of the paragraph.
Quote
q, a small quotation embedded within other content.
Blockquote
blockquote, a large, stand alone quote from another source.
Citation
cite, a citation for another source, often used with quotations. A person's name, a URL, a book, a movie title, etc.
